This repo contains a small react based chat application that uses websockets for real-time chatting.

## How to run
1. Clone this repo
2. `$ cd react-chat`
3. `$ npm install`
4. `$ npm run build`
5. `$ npm run server`
6. http://localhost:3001/

\* note `$` simply denotes a command prompt. It is not part of the command.
